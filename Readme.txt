Setup backend:
1- Change the connection string in appsettings.json file.
2- Run Update-Database command in package manager console by seleting Data project as default.
3- Build project and run the project

Setup frontend:
1- Open frontend project folder in VS code.
2- Run npm install command on terminal.
3- After step-2 is completed Run ng serve command.
4- Navigate to URL mentioned in terminal in my case it was http://localhost:4200/
